#   Heroes app
Simple app to showcase Marvel and DC heroes, built with React.

## Installation
1. Clone the repository
```bash
git clone https://github.com/Fernando-LRz/heroes-app.git
``` 
2. Go inside the project folder
```bash
cd heroes-app
```
3. Install dependencies
```bash
npm install
```

## Run the app
```bash
npm run dev
```
